## 🛠️ How to publish on GitHub Pages

1. Create a new repository on GitHub (e.g. `portfolio`)
2. Upload `index.html` and `README.md` to the repository
3. Go to **Settings → Pages**
4. Under **Source**, select `Deploy from a branch`
5. Choose branch `main` and folder `/ (root)`
6. Click **Save** — your site will be live in ~1 minute at:
   `https://<your-username>.github.io/<repo-name>/`

## ✏️ Customization

Open `index.html` in any text editor to update:
- Links for LinkedIn, GitHub, email
- CV download link (replace `href="#"` in the Download CV button with your PDF URL)
- Any personal details
